# Bubble-Typing-Game
A fun typing game where players pop bubbles by typing the letters inside them. The game features a windowed mode, random bubble generation, and a dynamic score system. Itâ€™s a simple yet engaging way to practice typing while avoiding falling bubbles!

## Features:
- Windowed mode with adjustable size
- Random bubbles with letters A-Z
- Typing to pop bubbles
- Score tracking
- Smooth animations
- Resizable window for different screen sizes

## How to Play:
1. Press the "Start" button to begin the game.
2. Bubbles with random letters will appear and float upwards.
3. Type the letters inside the bubbles to pop them.
4. Avoid letting the bubbles reach the top!
5. Your score increases every time you pop a bubble.
6. If a bubble reaches the top, the game ends.
